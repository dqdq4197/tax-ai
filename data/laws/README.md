# 세법 텍스트 파일

이 디렉토리에 법령 텍스트 파일을 배치한 뒤 `pnpm db:seed`를 실행하면 pgvector DB에 삽입됩니다.

## 폴더 구조

```
data/laws/
├── common/           # 공통 조항 — 모든 소득 유형에 적용 (세율, 기본공제 등)
│   ├── 소득세법.txt
│   └── 소득세법_시행령.txt
├── business/         # 사업소득 전용 조항
│   └── 사업소득_필요경비.txt
└── employment/       # 근로소득 전용 (추후 추가 시)
```

소득 유형 추가 시: 폴더 생성 + `scripts/seed-laws.ts`의 `FILES` 배열에 항목 추가.

## 다운로드 방법

[법령 다운로드 방법 안내](../../docs/law-download.md) 참고.

## 파일 형식

- 인코딩: **UTF-8**
- 형식: 법제처 원문 텍스트 (HTML 태그 없는 순수 텍스트)
- 각 조문은 `제N조(제목)` 형식으로 시작해야 청킹이 정상 동작합니다
