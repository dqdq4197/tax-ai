# 법령 텍스트 다운로드 방법

## 출처

**법제처 국가법령정보센터**: https://www.law.go.kr

> CLAUDE.md 규칙: 세법 데이터는 공식 출처만 허용. 블로그·뉴스 등 비공식 출처 금지.

---

## 다운로드 절차

### 1. 소득세법 / 소득세법 시행령

1. https://www.law.go.kr 접속
2. 검색창에 `소득세법` 입력 → 현행 법령 선택
3. 우측 상단 **다운로드** 버튼 클릭
4. **PDF** 또는 **텍스트(TXT)** 형식 선택
   - TXT를 선택하면 바로 사용 가능
   - PDF를 선택하면 `pdftotext -layout input.pdf output.txt` 로 변환 후 페이지 헤더/바닥글 제거 필요
5. UTF-8로 저장 확인 후 `data/laws/common/소득세법.txt`로 배치
6. 동일 방법으로 `소득세법 시행령` → `data/laws/common/소득세법_시행령.txt`

### 2. 사업소득 필요경비 (선택)

사업소득 필요경비 관련 조항(소득세법 제27조~제35조 등)만 필요하다면
전체 소득세법에서 해당 조문만 발췌하여 `data/laws/business/사업소득_필요경비.txt`로 저장.

---

## 텍스트 형식 요구사항

seed 스크립트의 청킹 로직은 **`제N조(제목)` 패턴**을 기준으로 조문을 분리합니다.

법제처 원문 텍스트는 이 형식을 따르므로 변환 과정에서 이 패턴이 깨지지 않도록 주의합니다.

**예시 (정상)**:

```
제27조(사업소득의 필요경비의 계산) ① 사업소득금액을 계산할 때 필요경비에 산입할 금액은...
```

**주의**: HTML 태그, 머리글/바닥글, 페이지 번호 등이 포함된 경우 제거 후 사용.

## 실행

```bash
# 파일 배치 후 실행
pnpm db:seed
```

기존 데이터는 삭제 후 재삽입되므로 법령 개정 시 파일 교체 후 재실행하면 됩니다.
